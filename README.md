# Web_Assignment_PartA



Το παρόν repository περιέχει την υλοποίηση του Part A της εργασίας για την υπηρεσία e-learning.  
Η εφαρμογή είναι πλήρως frontend και λειτουργεί μόνο στον browser χωρίς χρήση backend.

## Περιγραφή
Η εργασία υλοποιεί μια βασική πλατφόρμα e-learning με HTML, CSS και JavaScript.  
Ο χρήστης μπορεί να περιηγηθεί σε σελίδες μαθημάτων, βιβλίων, λεπτομερειών περιεχομένου και να
συμπληρώσει φόρμα εγγραφής.

Η δομή έχει οργανωθεί σύμφωνα με τις οδηγίες της εκφώνησης, με χωρισμένους φακέλους για:
- HTML σελίδες
- CSS αρχεία 
- JavaScript 
- assets και εικόνες

## Τρόπος εκτέλεσης
'Ανοιγμα του `index.html` σε οποιοδήποτε browser.

## Περιεχόμενα
- `index.html` – αρχική σελίδα της πλατφόρμας  
- `courses.html` – λίστα μαθημάτων  
- `books.html` – λίστα βιβλίων  
- `course-details.html` – σελίδα αναλυτικών πληροφοριών  
- `register.html` – φόρμα εγγραφής χρήστη  
- `about.html` – σελίδα μελών ομάδας  

## Τι περιλαμβάνει το Part A
- Πλήρως responsive σχεδιασμό (mobile, tablet, desktop)
- Κοινό μενού πλοήγησης σε όλες τις σελίδες
- Δυναμική εμφάνιση μαθημάτων και βιβλίων μέσω JavaScript
- Φίλτρα/επιλογές σε λίστες
- Επικύρωση φόρμας εγγραφής με HTML5 + JavaScript
- Οργάνωση κώδικα σε αρχεία σύμφωνα με τη δομή της εκφώνησης
